# Single level neural network via baritsu

[`mlpack::perceptron()`](https://rdrr.io/pkg/mlpack/man/perceptron.html)
is an implementation of a perceptron, which is a single level neural
network.

## Details

For this engine, there is a single mode: classification

### Tuning Parameters

This model has 1 tuning parameter:

- `epochs`: Maximum number of iterations (type: integer)
