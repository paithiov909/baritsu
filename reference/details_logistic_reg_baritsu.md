# Logistic regression via baritsu

[`mlpack::logistic_regression()`](https://rdrr.io/pkg/mlpack/man/logistic_regression.html)
is an implementation of L2-regularized logistic regression for two-class
classification.

## Details

For this engine, there is a single mode: classification

### Tuning Parameters

This model has 1 tuning parameter:

- `penalty`: Amount of regularization (type: double)
