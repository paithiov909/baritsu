# L2-regularized support vector machine via baritsu

[`mlpack::linear_svm()`](https://rdrr.io/pkg/mlpack/man/linear_svm.html)
is an implementation of linear SVM for multiclass classification.

## Details

For this engine, there is a single mode: classification

### Tuning Parameters

This engine has 1 tuning parameter:

- `margin`: Insensitivity Margin (type: double)
